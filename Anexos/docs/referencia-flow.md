# Referencia Flow

La carpeta `/Flow` contiene una página de referencia con una navegación y estructura visual que se usará como inspiración para la app ROSSI.

Esta referencia debe usarse únicamente para entender:
- estructura general de la navegación;
- organización visual de las pantallas;
- posibles secciones;
- jerarquía de contenido;
- comportamiento general del ecommerce.

No se debe copiar el código directamente.

La app final debe implementarse desde cero en React, dentro de la carpeta `/codigo`, usando únicamente las tecnologías permitidas para el obligatorio.

Si el código de `/Flow` usa tecnologías no permitidas, deben ignorarse.

Si hay contradicciones entre `/Flow`, el diseño de Figma, el análisis de marca y la letra del obligatorio, siempre se debe priorizar este orden:

1. Letra del obligatorio.
2. Restricciones técnicas de clase.
3. Diseño de Figma.
4. Análisis de marca ROSSI.
5. Flow de referencia.
